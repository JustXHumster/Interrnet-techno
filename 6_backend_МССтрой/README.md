Запуск с Docker
bash
# Сборка образа
docker build -t tree-store .

# Запуск контейнера
docker run --rm tree-store

API
TreeStore(items)
Конструктор класса. Принимает массив объектов с полями id и parent.

Параметры:

items - список словарей, каждый словарь должен содержать поля id и parent

getAll()
Возвращает исходный массив элементов.

Возвращает: список всех элементов

getItem(id)
Возвращает элемент по его идентификатору.

Параметры:

id - идентификатор элемента

Возвращает: словарь с данными элемента или None, если элемент не найден

getChildren(id)
Возвращает массив дочерних элементов для указанного ID.

Параметры:

id - идентификатор родительского элемента

Возвращает: список дочерних элементов (может быть пустым)

getAllParents(id)
Возвращает массив родительских элементов от указанного элемента до корня.

Параметры:

id - идентификатор элемента

Возвращает: список родительских элементов в порядке от ближайшего к корневому

Особенности реализации
Оптимизация производительности: все данные предварительно индексируются в словарях для O(1) доступа

Безопасность данных: методы возвращают копии внутренних структур, предотвращая случайные изменения

Эффективность: минимальное количество проходов по массиву при инициализации
