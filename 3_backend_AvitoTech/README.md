Описание
Библиотека предоставляет функцию get_matrix(), которая:

Асинхронно загружает квадратную матрицу с удаленного сервера по HTTP(S) или из локального файла

Парсит матрицу из текстового формата с разделителями

Выполняет обход матрицы по спирали против часовой стрелки

Возвращает список элементов в порядке обхода

Требования
Python 3.7 или выше

Для HTTP-запросов рекомендуется aiohttp, но библиотека имеет fallback на стандартные средства Python



#Рекомендуемые зависимости для асинхронных HTTP-запросов

pip install aiohttp
