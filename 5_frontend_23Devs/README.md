# Форма регистрации с валидацией

Проект представляет собой веб-форму регистрации с клиентской валидацией всех полей. Реализована на чистом HTML, CSS и JavaScript без использования сторонних библиотек.

## Функциональность

- **Валидация всех полей формы:**
  - Имя и фамилия (только буквы, дефисы и апострофы, длина 2-30 символов)
  - Email (проверка формата)
  - Пароль (минимум 8 символов, цифра, строчная буква, заглавная буква, спецсимвол)
  - Подтверждение пароля (совпадение с паролем)
  - Дата рождения (не в будущем, возраст 18+ лет)

- **Визуальная обратная связь:**
  - Подсветка полей (зеленая для валидных, красная для невалидных)
  - Сообщения об ошибках под каждым полем
  - Динамическое отображение требований к паролю
  - Сводка всех ошибок валидации

- **Удобство использования:**
  - Кнопки показа/скрытия пароля
  - Блокировка кнопки отправки до полной валидации
  - Автоматическая прокрутка к первому невалидному полю
  - Валидация при потере фокуса и при вводе

## Технологии

- HTML5
- CSS3 (адаптивный дизайн)
- JavaScript (ES6+)
